# PZ_7
1. **Что такое регулярные выражения (RegEx) и в каких задачах обработки текста они наиболее эффективны?**
   - Регулярные выражения - это последовательность символов, задающая шаблон для поиска и обработки текста
   - Наиболее эффективны для: поиска email, телефонов, дат; валидации ввода; замены текста по шаблону; разбора логов

2. **Какой модуль в Python необходимо импортировать для работы с регулярными выражениями?**
   - Модуль re (import re)

3. **В чём разница между методами re.search() и re.findall()? Какой из них возвращает список всех совпадений?**
   - re.search() - ищет первое совпадение и возвращает объект Match (или None)
   - re.findall() - ищет ВСЕ совпадения и возвращает список строк
   - re.findall() -  возвращает список всех совпадений

4. **Если вам нужно проверить, что строка начинается с определённого паттерна, какой метод модуля re вы будете использовать?**
   - re.match() - проверяет совпадение с начала строки
   - Или re.search() с метасимволом `^` в начале паттерна

5. **Что означает метасимвол . (точка) в регулярных выражениях?**
   - Точка означает любой одиночный символ (кроме символа новой строки \n)

6. **Что такое группировка в регулярных выражениях и какой синтаксис используется для её обозначения?**
   - Группировка - объединение частей шаблона для последующего использования
   - Синтаксис: круглые скобки ( )
   - Группы нумеруются по порядку открывающих скобок: \1, \2 и т.д.
   - Для групп без сохранения: (?: )
